## Overview

ExamFlow AI is a modern AI-powered exam preparation platform designed to help students plan efficiently, stay consistent, and maximize performance.

Users can upload a syllabus PDF or manually enter topics. The platform analyzes modules, priorities, and deadlines to generate a personalized study plan.

---

## Key Features

- AI syllabus analyzer  
- Personalized study planner  
- Focus timer with hour tracking  
- Crisis Mode for last-minute preparation  
- AI tutor assistance  
- Progress analytics dashboard  
- Goal milestones and reminders

## Requirements

Before starting, install:

- Node.js (Version 18 or above)
- npm
- Git (optional)

Check installation:

```bash
node -v
npm -v
```
Installation Guide
1. Clone Repository
git clone https://github.com/yourusername/examflow-ai.git
cd examflow-ai
2. Install Dependencies
npm install
3. Configure Environment Variables

Create a file named .env.local

GEMINI_API_KEY=your_api_key
VITE_FIREBASE_API_KEY=your_key
VITE_FIREBASE_AUTH_DOMAIN=your_domain
VITE_FIREBASE_PROJECT_ID=your_project_id
Run Project
npm run dev

Open in browser:

http://localhost:5173
Build for Production
npm run build
Deployment
Netlify

Upload the generated dist folder after running:

npm run build
Vercel

Connect your GitHub repository and deploy.
